# Proyek Big Data Kebugaran Jasmani Siswa

Proyek ini bertujuan untuk menganalisis data kebugaran jasmani siswa menggunakan pendekatan Big Data melalui Exploratory Data Analysis (EDA) dan visualisasi data.

## Dataset
Dataset berisi lebih dari 1000 record data kebugaran siswa yang mencakup:
- Push-up
- Lari 10 meter
- Sit-reach
- Skor dan kategori kebugaran
- Kehadiran Pendidikan Jasmani

Dataset telah melalui proses data cleaning dan transformasi.

## Metode
- Data Cleaning
- Data Transformation
- Exploratory Data Analysis (EDA)
- Visualisasi Data

## Tools
- Spreadsheet (Microsoft Excel / Google Sheets)
- GitHub

## Luaran
- Dataset bersih
- Visualisasi data
- Laporan akhir proyek (PDF)

Proyek ini dibuat untuk memenuhi tugas mata kuliah Big Data.
